# AI4BI Demo v3.5 – Decision Intelligence Demo

A portfolio-level **AI Decision Intelligence** dashboard built with **Streamlit + Plotly**.

This demo simulates a modern, simplified BI experience inspired by tools like PowerBI/Tableau:
- KPI monitoring
- interactive analytics
- risk heatmap analysis
- AI-style business insights
- rule-based decision recommendations

## Features

### KPI Overview
Top-line business KPIs using Streamlit metrics:
- Total Revenue
- Total Cost
- Profit
- Profit Margin

### Analytics
Interactive Plotly visuals:
1. Revenue by Category (bar chart)
2. Profit Trend (line chart)

### Risk Analysis
Computes `Risk Score = Cost / Revenue` and classifies records into:
- Low Risk
- Medium Risk
- High Risk

Displays a color-coded risk table for quick scanning.

### AI Insight
Panel titled **AI Business Insight** that highlights:
- Top performing category
- Highest cost driver
- Most profitable segment

### Decision Suggestions
Rule-based recommendations generated by a decision engine, such as:
- Reduce cost in high-cost categories
- Increase investment in top-profit categories
- Monitor high-risk segments

## Module Guide

- `app.py`  
  Main Streamlit entry point and page layout orchestration.

- `analytics.py`  
  Data loading, data preparation, KPI calculations, chart datasets, risk scoring, and AI insights.

- `decision_engine.py`  
  Generates business decision recommendations from analytics outputs.

- `ui_components.py`  
  Reusable dashboard UI blocks (KPI panel, charts, risk heatmap, insight/recommendation sections).

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

Then open the local Streamlit URL shown in your terminal.
